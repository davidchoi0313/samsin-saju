# 🌙 삼신이 사주풀이 (samsin-saju)

> 무게 잡지 않는 다정한 견습 도사 **삼신이**가, 네 사주 보따리를 펼쳐 다정히 읽어주는 Claude 플러그인.
> 운세로 겁주는 게 아니라, 오늘 하루 어깨를 토닥여주는 친구. ✨

---

## 삼신이가 누구냐면

삼신이는 **운명을 점지하는 삼신할미의 막내 견습생**이에요. 다정한데 가끔 깜빡깜빡하는 허당 도사랍니다.
무게 잡는 점쟁이가 아니라 옆집 손녀 같은 견습 도사예요 — "맞다/틀리다"가 아니라 *"이런 기운이 있단다, 참고만 하렴"* 하고 다정한 반말로 어깨를 토닥여 줍니다.

삼신이의 풀이는 두 층으로 동작해요.

1. **만세력 엔진 (결정론 산출)** — 생년월일·생시·성별을 받아 사주팔자·오행·십신·십이운성·대운·세운·신살을 *계산*합니다. 같은 입력이면 항상 같은 결과예요.
2. **삼신이 해석** — 그 산출 카드의 값만 근거로, 메뉴별 톤으로 다정히 풀어줍니다. 사주를 *지어내지* 않고, 이미 적힌 것을 읽어줄 뿐이에요.

## 설치 전 준비

본격적으로 설치하기 전에 아래 세 가지만 확인해 주세요.

- **실행 환경** — 삼신이는 **윈도우(및 맥) Claude 데스크톱 앱**에서 설치해 씁니다. 플러그인은 Claude 유료 플랜(Pro·Max·Team·Enterprise)에서 사용할 수 있어요. 플러그인이 번들한 스킬은 웹 채팅·데스크톱 앱 Chat 탭·Cowork에서 모두 쓸 수 있고, 일부 기능(서브에이전트·훅)은 **Cowork에서만** 동작합니다. 삼신이를 Cowork에서 쓰려면 Cowork 사용이 켜져 있어야 합니다.
- **공개 레포** — 삼신이는 **GitHub 공개 레포**로 배포됩니다.
- **만세력 엔진(Python)** — 풀이의 1층 엔진은 Python을 씁니다. `lunar_python` 패키지가 없으면 엔진이 자동 설치를 시도하고, 그래도 안 되면 삼신이가 설치 방법을 안내해 줍니다.

## 윈도우(및 맥) Claude 데스크톱 앱에서 설치하기

삼신이는 **데스크톱 앱의 화면(GUI)에서 메뉴를 눌러** 설치합니다. 터미널에 명령어를 칠 필요가 없어요. 아래 순서대로 따라 하면 됩니다.

> 아래 메뉴 명칭·경로는 Claude 공식 안내(Help Center, "Use plugins in Claude")를 기준으로 적었습니다. 앱 버전·언어 설정에 따라 화면의 실제 글자가 조금 다를 수 있으니, 같은 위치의 메뉴를 찾아 누르시면 됩니다.

### (a) 마켓플레이스 추가 — 삼신이가 사는 보따리 가게를 등록

1. 데스크톱 앱 왼쪽 사이드바에서 **Customize(커스터마이즈)** 메뉴를 엽니다. (Cowork에서 쓸 거라면 먼저 **Cowork** 탭을 연 뒤 Customize를 엽니다.)
2. **Plugins(플러그인)** 탭으로 이동합니다.
3. **Personal plugins(개인 플러그인)** 섹션에서 **`+` 버튼**을 누르고 **"Add marketplace(마켓플레이스 추가)"** 를 고릅니다.
4. **"Add from a repository(레포지토리에서 추가)"** 를 골라 GitHub 공개 레포를 동기화합니다. 레포 주소는 아래 형태예요.

```
https://github.com/davidchoi0313/samsin-saju
```

`이렇게 하면 삼신이 플러그인이 들어 있는 **마켓플레이스(보따리 가게)가 내 데스크톱 앱에 등록**됩니다.


### (b) 플러그인 설치 — 삼신이를 내 보따리로 데려오기

1. 같은 **Customize → Plugins** 화면에서 **"Browse plugins(플러그인 둘러보기)"** 를 누릅니다.
2. 목록에서 **`samsin-saju`** 를 찾아 **"Install(설치)"** 을 누릅니다.

방금 등록한 마켓플레이스에서 삼신이가 내 보따리로 들어옵니다.

### (c) 설치 확인 — 삼신이가 잘 왔는지 보기

설치가 끝났는지 확인하는 방법은 두 가지예요.

- **목록으로 확인** — Customize → Plugins 화면의 설치된 플러그인 목록에 **`samsin-saju`** 가 보이고 켜져(enabled) 있으면 자리를 잡은 거예요.
- **불러서 확인** — Cowork(또는 Chat) 대화창에 **"사주 봐줘"** 라고 입력해 보세요. 삼신이가 *"어서 오너라 아가…"* 하고 인사하며 보따리를 펼치면 정상 동작하는 겁니다.

> 설치 직후 첫 호출에서 만세력 엔진(Python)이 `lunar_python` 을 자동으로 준비할 수 있어요. 첫 풀이가 살짝 느릴 수 있는데, 그건 엔진이 처음 채비하는 시간이니 잠깐 기다려 주세요.

<details>
<summary><b>개발자용 참고 — 터미널 Claude Code에서 설치하기</b></summary>

데스크톱 앱이 아니라 터미널 **Claude Code**를 쓰는 개발자라면, 슬래시 명령으로도 설치할 수 있습니다. (일반 사용자는 위 데스크톱 앱 절차만 따르면 됩니다.)

## 사용법 — 삼신이를 부르는 법

설치가 끝나면 따로 명령어를 외울 필요 없이 **Cowork(또는 Chat) 대화창에 그냥 말 걸듯이 입력하면** 됩니다. 아래처럼요.

```
사주 봐줘
```

```
1990년 3월 5일 오전 7시 남자 평생사주 봐줘
```

```
올해 운세 알려줘
```

```
나랑 1988년생 토끼띠 궁합 봐줘
```

```
오늘의 운세 어때
```

삼신이는 풀이를 하려면 **생년월일·생시·성별·음양력(양력/음력)** 이 필요해요. 처음에 다 말하지 않아도 괜찮아요 — 빠진 게 있으면 삼신이가 *"몇 년 몇 월 며칠에 났니?"*, *"양력이냐 음력이냐?"*, *"태어난 시(時)는 아니?"* 하고 하나씩 다정히 되물어 채웁니다. (생시를 모르면 "모름"이라고 해도 풀어줘요. 일부 메뉴는 시 정보 없이도 봐줍니다.)

> 메뉴를 콕 집지 않고 "사주 봐줘"만 해도 됩니다. 그러면 삼신이가 우선 **평생사주**부터 펼치고, 다른 메뉴도 보고 싶은지 가볍게 여쭤봐요.

> 메뉴 15(인재 채용 참고)는 **합격/불합격·점수·등급을 매기는 도구가 아니에요.** 면접 때 곁들이는 *참고 보조자료*일 뿐이며, 채용 결정은 직무 자격·역량 평가가 언제나 우선입니다.

## 문제 해결 (FAQ)

**Q. 마켓플레이스 추가가 안 돼요 / 레포를 못 찾는대요.**
A. 두 가지를 확인해 주세요. ① 레포가 **공개(Public)** 상태인지 — 비공개면 추가가 막힙니다. ② 입력한 레포 주소의 **계정명과 레포명이 정확한지**. 오타나 대소문자가 다르면 못 찾아요. (브라우저에서 `https://github.com/<GitHub사용자명>/samsin-saju-marketplace` 가 실제로 열리는지로 검증할 수 있어요.) 그리고 입력 칸이 받는 형식이 전체 URL인지 짧은 형태(`<GitHub사용자명>/samsin-saju-marketplace`)인지 앱 안내에 따라 한 번씩 바꿔 시도해 보세요.

**Q. 설치는 됐는데 "사주 봐줘" 해도 삼신이가 안 나와요.**
A. **Customize → Plugins** 목록에 `samsin-saju` 가 **설치됨(enabled)** 으로 보이는지 먼저 확인해 주세요. 목록엔 있는데 꺼져 있다면 켜 주시고, 그래도 안 뜨면 데스크톱 앱을 한 번 재시작한 뒤 Cowork(또는 Chat) 대화창에서 다시 불러 보세요. (서브에이전트 등 일부 기능은 Cowork에서만 동작하니, 안 보이면 Cowork 탭에서 시도해 보시는 것도 방법이에요.)

**Q. 첫 풀이가 멈추거나 오류가 나요 / "만세력에 먼지가 끼었다"고 해요.**
A. 1층 만세력 엔진은 Python `lunar_python` 패키지를 씁니다. 첫 호출에서 자동 설치를 시도하는데, 환경에 따라 실패할 수 있어요. 그럴 땐 삼신이가 입말로 설치 방법을 안내해 줍니다 — 안내대로 패키지를 준비한 뒤 다시 "사주 봐줘" 해 보세요.

**Q. 풀이 결과가 매번 달라지거나 이상해요.**
A. 같은 생년월일·생시·성별·음양력을 넣었다면 엔진 산출(사주팔자·오행·대운 등)은 **항상 같아야** 정상입니다. 결과가 들쭉날쭉하면 입력값(특히 양력/음력, 생시)이 그때그때 다르게 전달됐을 가능성이 커요. 입력을 정확히 한 번 더 짚어 말해 주세요.
