## 삼신이가 누구냐면

삼신이는 **운명을 점지하는 삼신할미의 막내 견습생**이에요. 다정한데 가끔 깜빡깜빡하는 허당 도사랍니다.
무게 잡는 점쟁이가 아니라 옆집 손녀 같은 견습 도사예요 — "맞다/틀리다"가 아니라 *"이런 기운이 있단다, 참고만 하렴"* 하고 다정한 반말로 어깨를 토닥여 줍니다.

삼신이의 풀이는 두 층으로 동작해요.

1. **만세력 엔진 (결정론 산출)** — 생년월일·생시·성별을 받아 사주팔자·오행·십신·십이운성·대운·세운·신살을 *계산*합니다. 같은 입력이면 항상 같은 결과예요.
2. **삼신이 해석** — 그 산출 카드의 값만 근거로, 메뉴별 톤으로 다정히 풀어줍니다. 사주를 *지어내지* 않고, 이미 적힌 것을 읽어줄 뿐이에요.

## 설치 전 준비

본격적으로 설치하기 전에 아래 두 가지만 확인해 주세요.

- **실행 환경** — 삼신이는 **Claude Code** 또는 **Cowork** 환경에서 동작하는 플러그인입니다. 둘 중 하나가 설치되어 있어야 `/plugin` 명령을 쓸 수 있어요. (아직 없다면 Claude Code / Cowork 를 먼저 설치해 주세요.)
- **공개 레포** — 삼신이는 **GitHub 공개 레포**로 배포됩니다. 아래 설치 명령의 `<GitHub사용자명>` 자리에는 이 레포가 올라간 실제 계정명을 넣게 됩니다. (레포가 공개(Public) 상태여야 설치가 진행됩니다 — 비공개면 마켓플레이스 추가 단계에서 막혀요.)
- **만세력 엔진(Python)** — 풀이의 1층 엔진은 Python을 씁니다. `lunar_python` 패키지가 없으면 엔진이 자동 설치를 시도하고, 그래도 안 되면 삼신이가 설치 방법을 안내해 줍니다.

## 설치 3단계

아래 세 단계를 차례대로 따라 하면 됩니다. 각 명령은 Claude Code / Cowork 입력창에 그대로 입력하세요.

### (a) 마켓플레이스 추가 — 삼신이가 사는 보따리 가게를 등록

```
/plugin marketplace add <GitHub사용자명>/samsin-saju-marketplace
```

이 명령은 삼신이 플러그인이 들어 있는 **마켓플레이스(보따리 가게)를 내 Claude Code 에 등록**합니다. `<GitHub사용자명>` 자리에는 이 레포가 올라간 실제 GitHub 계정명을 넣어 주세요.

예시 (실제 공개 레포 주소로 바꿔주세요):

```
/plugin marketplace add your-id/samsin-saju-marketplace
```

### (b) 플러그인 설치 — 삼신이를 내 보따리로 데려오기

```
/plugin install samsin-saju@samsin-saju-marketplace
```

이 명령은 방금 등록한 마켓플레이스에서 **`samsin-saju` 플러그인을 실제로 설치**합니다. `@samsin-saju-marketplace` 는 "어느 가게의 삼신이인지"를 가리키는 출처 표시예요.

### (c) 설치 확인 — 삼신이가 잘 왔는지 보기

설치가 끝났는지 확인하는 방법은 두 가지예요.

- **목록으로 확인** — `/plugin` 을 입력하면 설치된 플러그인 목록이 나옵니다. 그 안에 **`samsin-saju`** 가 보이면 자리를 잡은 거예요.
- **불러서 확인** — 입력창에 **"사주 봐줘"** 라고 말해 보세요. 삼신이가 *"어서 오너라 아가…"* 하고 인사하며 보따리를 펼치면 정상 동작하는 겁니다.

> 설치 직후 첫 호출에서 만세력 엔진(Python)이 `lunar_python` 을 자동으로 준비할 수 있어요. 첫 풀이가 살짝 느릴 수 있는데, 그건 엔진이 처음 채비하는 시간이니 잠깐 기다려 주세요.

## 사용법 — 삼신이를 부르는 법

설치가 끝나면 따로 명령어를 외울 필요 없이 **그냥 말 걸듯이 부르면** 됩니다. 아래처럼요.

```
사주 봐줘
```

```
1990년 3월 5일 오전 7시 남자 평생사주 봐줘
```

```
올해 운세 알려줘
```

```
나랑 1988년생 토끼띠 궁합 봐줘
```

```
오늘의 운세 어때
```

삼신이는 풀이를 하려면 **생년월일·생시·성별·음양력(양력/음력)** 이 필요해요. 처음에 다 말하지 않아도 괜찮아요 — 빠진 게 있으면 삼신이가 *"몇 년 몇 월 며칠에 났니?"*, *"양력이냐 음력이냐?"*, *"태어난 시(時)는 아니?"* 하고 하나씩 다정히 되물어 채웁니다. (생시를 모르면 "모름"이라고 해도 풀어줘요. 일부 메뉴는 시 정보 없이도 봐줍니다.)

> 메뉴를 콕 집지 않고 "사주 봐줘"만 해도 됩니다. 그러면 삼신이가 우선 **평생사주**부터 펼치고, 다른 메뉴도 보고 싶은지 가볍게 여쭤봐요.

## 15메뉴, 이렇게 부르면 돼요

각 메뉴를 어떤 말로 부르면 되는지 예시 발화를 한 줄씩 정리했어요. (예시 속 생년월일·성별은 자유롭게 바꿔 말하면 됩니다.)

| # | 메뉴 | 이렇게 불러보세요 (예시 발화) |
|---|---|---|
| 01 | 평생사주 | "1990년 3월 5일 오전 7시 남자 평생사주 봐줘" |
| 02 | 올해운세 | "올해 운세 알려줘" / "2026 신년운세 봐줘" |
| 03 | 궁합 | "나랑 1988년생 토끼띠 궁합 봐줘" / "우리 둘 잘 맞아?" |
| 04 | 귀신사주 | "내 귀신사주 봐줘" |
| 05 | 전생사주 | "내 전생사주 풀어줘" |
| 06 | 용신사주 | "내 용신이 뭔지 봐줘" |
| 07 | 직업운 | "나한테 맞는 직업운 봐줘" |
| 08 | 결혼·연애운 | "내 결혼운 어때" / "연애운 봐줘" |
| 09 | 토정비결 | "올해 토정비결 봐줘" |
| 10 | 신살풀이 | "내 사주 신살풀이 해줘" |
| 11 | 대운풀이 | "내 대운 풀어줘" / "대운 흐름 봐줘" |
| 12 | 오늘의운세 | "오늘의 운세 어때" / "오늘 운세 봐줘" |
| 13 | 건강운 | "내 건강운 봐줘" |
| 14 | 사업 궁합 | "1985년생 동업자랑 사업 궁합 봐줘" / "동업 궁합 봐줘" |
| 15 | 인재 채용 참고 | "지원자 성향 참고로 봐줘" / "같이 일하면 어떨까 참고만 봐줘" |

> 메뉴 14·15는 **두 사람 정보**가 필요해요(나 + 상대). 상대 정보를 넣을 때는 상대의 동의를 전제로 가볍게 봐주는 *참고용*이라는 점, 삼신이가 먼저 짚어 줍니다.
> 메뉴 15(인재 채용 참고)는 **합격/불합격·점수·등급을 매기는 도구가 아니에요.** 면접 때 곁들이는 *참고 보조자료*일 뿐이며, 채용 결정은 직무 자격·역량 평가가 언제나 우선입니다.

## 문제 해결 (FAQ)

**Q. `/plugin marketplace add` 가 안 돼요 / 레포를 못 찾는대요.**
A. 두 가지를 확인해 주세요. ① 레포가 **공개(Public)** 상태인지 — 비공개면 추가가 막힙니다. ② 명령의 경로가 `<GitHub사용자명>/samsin-saju-marketplace` 형태로 **계정명과 레포명이 정확한지**. 오타나 대소문자가 다르면 못 찾아요. (브라우저에서 `https://github.com/<GitHub사용자명>/samsin-saju-marketplace` 가 실제로 열리는지로 검증할 수 있어요.)

**Q. 설치는 됐는데 "사주 봐줘" 해도 삼신이가 안 나와요.**
A. `/plugin` 목록에 `samsin-saju` 가 **설치됨(enabled)** 으로 보이는지 먼저 확인해 주세요. 목록엔 있는데 꺼져 있다면 켜 주시고, 그래도 안 뜨면 Claude Code / Cowork 를 한 번 재시작한 뒤 다시 불러 보세요.

**Q. 첫 풀이가 멈추거나 오류가 나요 / "만세력에 먼지가 끼었다"고 해요.**
A. 1층 만세력 엔진은 Python `lunar_python` 패키지를 씁니다. 첫 호출에서 자동 설치를 시도하는데, 환경에 따라 실패할 수 있어요. 그럴 땐 삼신이가 입말로 설치 방법을 안내해 줍니다 — 안내대로 패키지를 준비한 뒤 다시 "사주 봐줘" 해 보세요.

**Q. 풀이 결과가 매번 달라지거나 이상해요.**
A. 같은 생년월일·생시·성별·음양력을 넣었다면 엔진 산출(사주팔자·오행·대운 등)은 **항상 같아야** 정상입니다. 결과가 들쭉날쭉하면 입력값(특히 양력/음력, 생시)이 그때그때 다르게 전달됐을 가능성이 커요. 입력을 정확히 한 번 더 짚어 말해 주세요.

## ⚠️ 꼭 읽어주세요 — 오락·참고용 안내

> 본 플러그인의 모든 풀이는 **전통 명리학·무속에 기반한 오락·참고용**이며,
> **의학·법률·재정적 조언이 아닙니다.**
> 중요한 결정은 본인의 판단과 전문가의 조언에 따르시기 바랍니다.

- 건강은 양생 습관까지만 안내하며 **질병을 진단하지 않습니다.** 증상이 있으면 전문의와 상담하세요.
- 재물·투자·계약·이직·채용 등 **중대한 결정의 근거로 삼지 마세요.**
- 인재 채용 참고(메뉴 15)는 **채용 결정·합격 판정·선별 도구가 아닙니다.** 나이·성별·출신 등을 이유로 한 차별은 법으로 금지됩니다.
- 삼신이는 운명을 단정하거나 겁주지 않아요. *"운명은 정해진 게 아니라 바꿀 수 있다"* 는 톤을 지킵니다.
