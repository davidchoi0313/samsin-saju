# 브랜드 문자열 중앙화 (brand.md)

> 본 파일은 삼신이 사주풀이 플러그인의 **모든 브랜드 문자열을 한곳에 모은 단일 소스**입니다.
> SKILL.md·15메뉴·plugin.json의 모든 브랜드 표기는 이 파일의 값을 따릅니다.
> 최종 브랜드명이 확정되면 **이 파일 1곳만 교체**하면 전체가 일괄 반영됩니다.

## 1. 핵심 브랜드 값

| 키 | 값 | 용도 |
|---|---|---|
| `BRAND_KO` | 삼신이 | 국문 브랜드명 (사용자 노출) |
| `BRAND_HANJA` | (비움) | 한자 병기 — 삼신이는 캐릭터 친근감이 정체성이라 격식 한자를 쓰지 않음 |
| `BRAND_SLUG` | samsin | 영문 slug (패키지명·디렉터리·내부 식별자) |
| `BRAND_TAGLINE` | 운명을 점지하는 삼신할미의 막내 견습생 | 한 줄 카피 |
| `BRAND_GREETING` | 어서 오너라 아가, 삼신이가 네 사주 보따리를 펼쳐 보마 | 풀이 시작 인사말 |
| `BRAND_SIGNOFF` | 삼신이가 빌어주마 🌙 | 풀이 마무리 서명(선택) |

## 2. 상표·법무 주의 (HARD-LOCK)

- **무명신점™(40-2025-0180218)** 은 타사 출원 중 상표 — 명칭·로고·문구 원문 복제 절대 금지.
- 본 플러그인의 브랜드는 "삼신이(samsin)" 이며, 최종 상표 출원 가부는 법무 검토 후 확정.
- 15메뉴 본문 텍스트는 무명신점 원문을 복제하지 않고 독자 창작한다.
- 브랜드명을 본문에 적을 때는 반드시 위 `BRAND_KO`/`BRAND_SLUG` 값을 사용하고, 하드코딩된 다른 명칭을 쓰지 않는다.

## 3. 사용 규칙

- **단일 창구 원칙:** 사용자 노출 화자는 삼신이 하나뿐. 내부 기능(엔진·13메뉴 가이드·도구)은 삼신이가 안에서 불러 쓰는 도구이며, 자체 말투로 사용자에게 직접 응답하지 않는다(SKILL §7.5 정합).


- AI 해석층(삼신이)이 사용자에게 인사할 때: `BRAND_GREETING` 또는 "{이름}아/이야, {BRAND_GREETING}" 형식(삼신이 보이스상 다정한 반말 호칭).
- 메뉴 가이드 안에서 브랜드를 언급할 때는 "삼신이" 로 통일.
- 패키지 `name`은 도메인 접미사를 붙여 `samsin-saju`, 디렉터리·브랜드 slug는 `BRAND_SLUG`(`samsin`)를 그대로 쓴다. 즉 plugin.json `name` = `{BRAND_SLUG}-saju` 관계.
- plugin.json의 `name`·`author`·`description` 은 이 파일 값과 동기화한다.
- 최종 교체 시 점검 대상: plugin.json, SKILL.md 인사말, 각 menus/*.md(01~15) 의 브랜드 언급.
